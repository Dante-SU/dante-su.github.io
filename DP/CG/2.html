回环检测

回环检测，又称闭环检测，是指机器人识别曾到达某场景，使得地图闭环的能力。说的简单点，就是机器人在左转一下，右转一下建图的时候能意识到某个地方是“我”曾经来过的，然后把此刻生成的地图与刚刚生成的地图做匹配。

Loss Function for neural rendering

1

$$arg\ \min_G \mathcal{L}(G(x),y)$$

while $G$ here means Generator and $\mathcal{L}$ here means loss function

2

What's a good loss function?

  • Capture realism
  • Enforce correspondence
  • Useful for many tasks
  • Adapt to new task/data

3

Common loss function for neural rendering

  • perceptual/feature loss
  • learnable loss(GAN's loss)
  • cycle-consistency loss
  • patchwise contrastive loss
  • style loss
  • contextual loss
  • domain-specific loss
  • 3D-aware loss
