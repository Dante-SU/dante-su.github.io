## 表格

### markdown方法

**一般情况**

```md
| 表头1 | 表头2 | 表头3 |
| :- | :-: | -: |
| 信息1 | 信息2 | 信息3 |
```

效果：

| 表头1 | 表头2 | 表头3 |
| :- | :-: | -: |
| 信息11111111 | 信息22222222 | 信息33333333 |

---

**格内换行**

```md
| 表头1 | 表头2 |
| :-: | :-: |
| 信息1 | 信息2<br>信息3 |
```

效果：

| 表头1 | 表头2 |
| :-: | :-: |
| 信息1 | 信息2<br>信息3 |


### html方法

**一般情况**

```html
<table>
<tr>
  <th>表头1</th>
  <th>表头2</th>
</tr>
<tr>
  <td rowspan="2">单元格信息1</td>
  <td>单元格信息2-1</td>
</tr>
<tr>
  <td>单元格信息2-2</td>
</tr>
</table>
```
效果：

<table>
<tr>
  <th>表头1</th>
  <th>表头2</th>
</tr>
<tr>
  <td rowspan="2">单元格信息1</td>
  <td>单元格信息2-1</td>
</tr>
<tr>
  <td>单元格信息2-2</td>
</tr>
</table>

---

**更改对齐方式**

```html
<table>
<tr  align="right">
  <th>表头1</th>
  <th>表头2</th>
  <th align="left">表头3</th>
</tr>
<tr>
  <td align="right">单元格信息1</td>
  <td align="center">单元格信息2</td>
  <td align="left">单元格信息3</td>
</tr>
</table>
```

效果：

<table>
<tr  align="right">
  <th>表头1</th>
  <th>表头2</th>
  <th align="left">表头3</th>
</tr>
<tr>
  <td align="right">单元格信息1</td>
  <td align="center">单元格信息2</td>
  <td align="left">单元格信息3</td>
</tr>
</table>

---

**表头**

```html
<table>
<caption> 我是表头 </caption>
</table>
```

效果：

<table>
<caption> 我是表头 </caption>
</table>

```html

  <td>单元格信息2-1</td>
</tr>
<tr>
  <td>单元格信息2-2</td>
</tr>
</table>
```

效果：

<table>
<tr>
  <th>表头1</th>
  <th>表头2</th>
</tr>
<tr>
  <td rowspan="2">单元格信息1</td>
  <td>单元格信息2-1</td>
</tr>
<tr>
  <td>单元格信息2-2</td>
</tr>
</table>

---

**一格多列**

```html
<table>
<tr>
  <th>表头1</th>
  <th>表头2</th>
  <th>表头3</th>
</tr>
<tr>
  <td>单元格信息1</td>
  <td colspan="2">单元格信息2</td>
</tr>
</table>
```

效果：

<table>
<tr>
  <th>表头1</th>
  <th>表头2</th>
  <th>表头3</th>
</tr>
<tr>
  <td>单元格信息1</td>
  <td colspan="2">单元格信息2</td>
</tr>
</table>
