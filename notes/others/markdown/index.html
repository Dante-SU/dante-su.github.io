Maths symbols and functions

Tag of functions

To name the functions with tag like below, we could us \tag{1.1} or \begin{equation} & \end{equation}

$$ x = y \tag{1.1} $$

Example:

$$ x = y \tag{1.1} $$

$$ \begin{equation}
  x = y
\end{equation} $$

Example:

$$ \begin{equation} x = y \end{equation} $$

(As this website is build by hugo and the style file has not listed this rule, so the effect couldn’t be show well.)

| :-: | :-: |
| Content_1 | Content_2 <br> Content_3 |

</tr>
</table>
