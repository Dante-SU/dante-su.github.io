ICCV

Tex2shape

From: ICCV'2019

homepage | paper | code

Background

Generate 3D human body model with details of garment from an RGB image(without depth information)

Motivation

Regression from image pixels directly to 3D mesh displacements is not so good:

  1. Ignoring the rich illumination and shading information contained in RGB values
  2. inputs and outputs are not aligned

Novelity

  1. propose to regress shape as UV-space displacement and normal map
  2. turn a hard full-body shape reconstruction problem into an easier 3D pose-independent image-to-image translation problem

Cons

  1. the photo must taken from the front
  2. pose is strictly to A pose
